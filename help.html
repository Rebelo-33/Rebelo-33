Help - Secret Gift Exchange

How It Works

  • Create a new list and add participant names.
  • Protect your list with a 4-digit PIN and a secret code (for organiser access).
  • Share the access link with participants.
  • Participants enter the list name and PIN to view and draw names.
  • The organiser uses the secret code to manage or edit the list.
