How It Works

  1. Create a new list and add participant names.
  2. Protect your list with a 4-digit PIN and a secret code.
  3. Share the list name and PIN with participants.
  4. Participants draw names by entering the list name and PIN.
  5. Organisers manage lists using the secret code.

Frequently Asked Questions

Can I participate and organise?

Yes! You can draw a name just like any participant.

Can I draw twice?

No. Each participant can only draw once. The result is saved on your browser.

I forgot my PIN or Secret Code. What should I do?

Unfortunately, these cannot be recovered. Ask the organiser to resend or create a new list.
