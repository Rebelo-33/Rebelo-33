Navigation & Usage Instructions

  • Create a New List: Click this to add participant names and save your gift exchange group. After saving, you will get a shareable link.
  • Go to Draw Page: Becomes active after a list is saved. Use it to draw a name from the saved list.
  • My Lists: Allows organizers to revisit their saved list by entering a list code and PIN.
  • Draw Name: Available to anyone with the shared link. Click to receive a randomly assigned name from the list. This can only be done once per participant.
  • View Drawn Name: After drawing, the same name is visible whenever the page is revisited.
  • Language Selector: Select a language to translate the entire site interface.
