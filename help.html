Help

Frequently Asked Questions

  • Can I participate and organise? Yes! You can draw a name just like any participant.
  • Can I draw twice? No. Each participant can only draw once, and the result is saved on your browser.
  • I forgot my PIN or Secret Code. Unfortunately, these cannot be recovered for security. You may create a new list or ask the organiser to resend it to you.

How It Works

  • Create a new list and add participant names.
  • Protect your list with a 4-digit PIN and a secret code (for organiser access).
  • Share the access link with participants.
  • Participants enter the list name and PIN to view and draw names.
  • The organiser uses the secret code to manage or edit the list.
