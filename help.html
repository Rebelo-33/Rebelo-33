How It Works

  1. Create a new list and add participant names.
  2. Protect your list with a 4-digit PIN and a secret code.
  3. Share the list name and PIN with participants.
  4. Participants draw names by entering the list name and PIN.
  5. Organisers manage lists using the secret code.
