📖 Help & Instructions

  • Create a List: Click “Create a New List”, add unique names, then enter a unique list name, 4-digit PIN, and secret code. The link will be generated for participants.
  • Access a List: Click “My Lists” or “Go to Draw Page”, then enter the correct list name and 4-digit PIN.
  • Draw a Name: Once inside the list, click “Draw Name”. You can only draw once, and your result is stored locally.
  • Copy & Share: Use the generated link to share the list with participants.
  • Language: Change language using the dropdown at the top.
  • Manage: Only the creator with the correct secret code can delete or modify the list in future updates.
