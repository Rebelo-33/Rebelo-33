Website Navigation Instructions

  • Create a New List: Start a list, add names, and protect it with a PIN.
  • Save List: Click Save to store your list to Firebase.
  • Copy Link: Share the generated URL via WhatsApp or Email.
  • Draw Name: Visit the shared link, enter the PIN, and draw a name (once).
  • View Drawn Name: Revisit the link to see your match again.
  • My Lists: Lets you access lists you've created from the same device.
  • Help: Access this guide from any page.
